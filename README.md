# TaxGo - Malaysia E-Invoice Platform

Professional website for TaxGo, Malaysia's comprehensive LHDN e-invoice compliance platform by BlueAura Technology Sdn Bhd.

## Features

- Fully responsive design
- SEO optimized with meta tags and structured data
- Professional color scheme matching BlueAura branding
- Interactive FAQ accordion
- Smooth scrolling navigation
- Contact form
- Mobile-friendly hamburger menu

## File Structure

```
taxgowebsite/
├── index.html          # Main HTML file
├── styles.css          # Stylesheet with Blue Aura color theme
├── script.js           # JavaScript for interactive features
├── sitemap.xml         # SEO sitemap
├── robots.txt          # Search engine crawler instructions
└── images/             # Image assets folder (to be added)
```

## Setup Instructions

1. **Add Images**: Create an `images` folder and add the following assets:
   - `taxgo-logo.svg` - TaxGo logo
   - `hero-dashboard.png` - Dashboard screenshot
   - `taxgo-og-image.jpg` - Open Graph image for social sharing (1200x630px)
   - `taxgo-twitter-image.jpg` - Twitter card image (1200x600px)
   - `favicon.ico` - Website favicon
   - `apple-touch-icon.png` - Apple touch icon (180x180px)

2. **Update URLs**: When deploying, update the canonical URLs in:
   - `index.html` (meta tags)
   - `sitemap.xml`
   - `robots.txt`

3. **Configure Analytics**: Add your Google Analytics or other tracking code in the `<head>` section of `index.html`

4. **Backend Integration**: Connect the contact form to your backend API or email service by updating the form submission handler in `script.js`
